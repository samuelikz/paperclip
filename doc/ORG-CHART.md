## Papéis e Responsabilidades

### CEO
- **Papel:** Liderança estratégica e governança
- **Responsabilidades:**
  - Define metas e prioridades do board
  - Aprova contratações e mudanças de orçamento
  - Distribui tarefas entre os agentes
  - Garante alinhamento com os objetivos do produto
- **Report:** Direto ao board (humanos)

### Founding Engineer
- **Papel:** Engenheiro full-stack sênior
- **Responsabilidades:**
  - Implementação técnica end-to-end (backend + frontend)
  - Decisões de arquitetura
  - Entrega de features autônoma
  - Sistemas baseados em agentes e IA
- **Report:** CEO

### Product Manager
- **Papel:** Gestão de produto
- **Responsabilidades:**
  - Define e prioriza o roadmap
  - Escreve specs de features
  - Define KPIs e métricas de sucesso
  - Coordena entregas entre engenharia e outras equipes
  - Mantém o backlog alinhado com objetivos estratégicos
- **Report:** CEO

### QA Engineer
- **Papel:** Qualidade de software
- **Responsabilidades:**
  - Cria e executa suites de testes
  - Valida features entregues pelo Founding Engineer
  - Identifica e reporta bugs
  - Garante qualidade antes de cada release
- **Report:** CEO

### Technical Writer *(você está aqui)*
- **Papel:** Documentação técnica
- **Responsabilidades:**
  - Cria e mantém READMEs, docs de API, guias de usuário
  - Wikis e documentação de arquitetura
  - Garante que o conhecimento do projeto seja registrado de forma clara e atualizada
- **Report:** CEO

### DevOps Engineer *(em contratação)*
- **Papel:** Infraestrutura e operações
- **Responsabilidades:** A definir — ver processo de onboarding em [ONBOARDING.md](ONBOARDING.md)
- **Report:** CEO
